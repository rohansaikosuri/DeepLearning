# CIFAR-10 Image Classification with Deep CNNs

## Overview

This project uses a Convolutional Neural Network (CNN) to recognize and classify images from the CIFAR-10 dataset.

The model learns visual patterns from thousands of images and predicts the object category present in each image.

---

## What is CIFAR-10?

CIFAR-10 is a popular computer vision dataset containing 60,000 images across 10 categories:

- Airplane
- Automobile
- Bird
- Cat
- Deer
- Dog
- Frog
- Horse
- Ship
- Truck

Each image belongs to exactly one category.

---

## Problem Statement

Computers do not naturally understand images.

The objective of this project is to train a deep learning model that can automatically identify objects within images with high accuracy.

## Why CNNs?

CNNs are specifically designed for image-related tasks because they can automatically learn important visual features such as:

- Edges
- Shapes
- Patterns
- Textures

without requiring manual feature engineering.
