# Deep Learning Projects

A collection of deep learning and machine learning projects developed as part of my academic and personal learning journey.

These projects explore different areas of Artificial Intelligence including:

- Healthcare Prediction
- Computer Vision
- Time Series Forecasting

Each project focuses on solving a real-world problem using data-driven approaches and neural network models.

---

## Projects Included

### 🩺 Diabetes Prediction Using a Custom Perceptron

Predicts whether a patient is likely to have diabetes based on medical measurements.

**Topics Covered**

- Data Cleaning
- Feature Engineering
- Classification
- Perceptron Algorithm

➡️ [Open Project](./diabetes-prediction-custom-perceptron)

---

### 🖼️ CIFAR-10 Image Classification with Deep CNNs

A Convolutional Neural Network trained to recognize objects from images.

**Topics Covered**

- Computer Vision
- Image Classification
- CNN Architecture
- Hyperparameter Tuning

➡️ [Open Project](./cifar10-image-classification-cnn)

---

### 📈 Google Stock Price Prediction using RNN

Uses Recurrent Neural Networks to learn patterns in historical stock prices and forecast future values.

**Topics Covered**

- Time Series Analysis
- Sequence Modeling
- RNN Networks
- Financial Forecasting

➡️ [Open Project](./google-stock-price-prediction-rnn)
