# Diabetes Prediction Using a Custom Perceptron

## Overview

This project predicts whether a patient is likely to have diabetes based on medical measurements such as glucose level, blood pressure, BMI, and age.

The goal was to understand the fundamentals of machine learning by building and training a Perceptron-based classification model while also performing data cleaning and preprocessing.

---

## Problem Statement

Diabetes is a common health condition that affects millions of people worldwide.

Healthcare providers often collect patient information that can help identify individuals at risk. This project explores whether machine learning can use these medical measurements to predict diabetes outcomes.

---

## Dataset

The dataset contains patient health information including:

- Glucose Level
- Blood Pressure
- BMI (Body Mass Index)
- Insulin Level
- Age
- Pregnancies
- Diabetes Pedigree Function

Target Variable:

- 0 = No Diabetes
- 1 = Diabetes

---

## Project Workflow

1. Data Cleaning
   - Removed invalid and missing values
   - Handled outliers

2. Data Preprocessing
   - Feature scaling and normalization
   - Train-test split

3. Model Development
   - Implemented a Perceptron model
   - Trained on historical patient data

4. Model Evaluation
   - Accuracy
   - Precision
   - Recall
   - Confusion Matrix
